🧩 STEP 1 — Express ko Vercel-compatible banao
❌ Galat (normal express)
app.listen(3000)
✅ Sahi (serverless)

Install:

bun add serverless-http

Phir:

import express from "express"
import serverless from "serverless-http"

const app = express()

app.get("/", (req, res) => {
  res.json({ message: "Hello from Vercel" })
})

export const handler = serverless(app)

👉 Ye required hai kyunki Vercel functions handler export mangta hai

🧩 STEP 2 — Folder structure sahi karo

Backend folder ke andar:

backend/
  api/
    index.ts   <-- entry point (IMPORTANT)
  package.json
  tsconfig.json

👉 Vercel automatically api/ folder ko functions bana deta hai

🧩 STEP 3 — api/index.ts
import { handler } from "../src/app" // ya jahan tumhara express hai

export default handler
🧩 STEP 4 — Bun config add karo

Root ya backend mein vercel.json:

{
  "bunVersion": "1.x"
}

👉 Ye official requirement hai

🧩 STEP 5 — Prisma setup fix karo (VERY IMPORTANT)
1. package.json mein:
{
  "scripts": {
    "postinstall": "prisma generate"
  }
}

👉 warna deploy pe Prisma client nahi banega

2. ENV variable

Vercel dashboard mein add karo:

DATABASE_URL=your_db_url

👉 bina iske Prisma fail ho jayega

3. Monorepo case (tumhara case)

Agar Prisma packages/db mein hai:

👉 ensure karo:

backend us package ko import kare

aur build ke waqt accessible ho

Example:

import { prisma } from "@repo/db"

  }

Build Command: bun run build
